# v66 Blue Design System Rebuild

تم تنفيذ إعادة بناء طبقة التصميم من الصفر باللون الأزرق، مع الحفاظ على هيكل الصفحات والراوتات والكروت والوظائف.

## ما تم تغييره
- استبدال `app/globals.css` بالكامل بطبقة تصميم جديدة.
- استبدال `app/landing.css` بطبقة landing زرقاء فقط.
- حذف آثار الألوان القديمة: الأخضر، الذهبي، mint، emerald، teal.
- توحيد متغيرات التصميم حول الأزرق:
  - `--brand`
  - `--brand-2`
  - `--brand-3`
  - `--brand-soft`
- الحفاظ على:
  - هيكل الصفحات.
  - كروت المشاريع.
  - صفحة الريلز.
  - لوحة الإدارة.
  - لوحات المستخدم.
  - الراوتات.

## ملاحظات
- لا يحتاج SQL.
- هذا تعديل تصميم فقط.
- السعر بقي باللون الأحمر حسب الطلب السابق.
