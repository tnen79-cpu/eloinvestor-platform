# EloInvestor v46 — Translation Full Coverage

## المطلوب بعد رفع النسخة
1. شغّل الملف:
   `SUPABASE_V46_TRANSLATION_FULL_COVERAGE.sql`
2. من لوحة الإدارة افتح: **اللغات والترجمات**.
3. اضغط **إضافة النصوص الافتراضية** مرة واحدة إذا أردت مزامنة النصوص من الكود إلى قاعدة البيانات.
4. عدّل أي نص من لوحة الإدارة، ثم حدّث الصفحة.

## ما تم تطويره
- ربط نماذج الدخول والتسجيل بجدول الترجمات.
- ربط صفحة إضافة المشروع ورسالة الترويج بعد الإضافة بجدول الترجمات.
- ربط كروت المشاريع، نصوص المخاطر/الطلب/المشاهدات، والتفاصيل.
- ربط صفحة الفرص والفلترة الأساسية بجدول الترجمات.
- ربط صفحات نجاح/فشل الدفع بجدول الترجمات.
- إضافة API داخلي للترجمة التلقائية:
  `/api/admin/translate`
- الزر التلقائي في الإدارة صار يستخدم مزود الترجمة إن تم تفعيله، أو fallback محلي عند عدم وجود مفتاح.

## ملاحظات مزود الترجمة
جدول `translation_provider_settings` جاهز. يمكن استخدام:
- `manual` بدون مفاتيح.
- `deepl` مع api_url و api_key.
- `azure` مع api_url و api_key.

لـ Azure ضع رابط endpoint الخاص بالترجمة بدون query، مثل:
`https://api.cognitive.microsofttranslator.com/translate`

لـ DeepL ضع رابط API المناسب لحسابك.
