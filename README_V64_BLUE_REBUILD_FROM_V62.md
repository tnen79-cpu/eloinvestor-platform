# v64 Blue Rebuild From v62

هذه النسخة مبنية على v62، وليس v63.

## ما تم
- إعادة ضبط هوية المنصة بالكامل إلى اللون الأزرق.
- إزالة آثار اللون الأخضر قدر الإمكان من CSS وTailwind classes.
- الحفاظ على هيكل الصفحات والراوتات والكروت والمشاريع كما في v62.
- تحسين وضوح النصوص والأزرار.
- إصلاح حالة التبويب النشط في لوحة الإدارة حتى يظهر بوضوح.
- السعر يظهر باللون الأحمر.
- تحسينات موبايل عامة بدون تغيير بنية الصفحات.

## ملاحظة
لا يحتاج SQL جديد؛ التعديل تصميم فقط.
