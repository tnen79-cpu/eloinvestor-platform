# v58 Header + Admin Refresh

## التعديلات

- إزالة مربع البحث من الهيدر العام بالكامل.
- إبقاء البحث والفلترة في الهيرو فقط.
- تحسين لوحة الإدارة بصريًا باتجاه حديث قريب من لوحات eCommerce/Admin مثل Paces و Vuexy:
  - Sidebar داكن ومنظم.
  - Topbar sticky.
  - Cards أخف وأوضح.
  - Inputs وButtons موحدة.
  - تحسين الجداول والتباعد.
  - Responsive أفضل للموبايل.

- لم يتم تغيير منطق الإدارة أو صلاحياتها، فقط ترتيب/ستايل وتجربة الاستخدام.
- لم أتمكن من تشغيل lint داخل البيئة لأن `node_modules` غير موجودة.
