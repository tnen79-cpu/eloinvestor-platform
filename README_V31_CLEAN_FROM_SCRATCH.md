# EloInvestor v31 — Clean UI rebuild from scratch

هذه النسخة تعيد بناء التصميم من الصفر بدل البناء فوق التصميم السابق.

## ما تم تغييره
- استبدال `app/globals.css` بالكامل بهوية تصميم واحدة نظيفة.
- حذف/إلغاء ملفات الواجهة القديمة غير المستخدمة مثل السلايدر والدكات العائمة القديمة.
- حذف شريط الأكشن العائم الذي كان يظهر في يمين الشاشة على الديسكتوب.
- تثبيت الهيدر، الصفحة الرئيسية، صفحة الفرص، كروت المشاريع، ولوحة المستخدم بهوية واحدة.
- عدم إضافة مشاريع أو بيانات وهمية؛ أي مشاريع تظهر هي من Supabase فقط.
- الإبقاء على Placeholder فقط عند عدم وجود صورة للمشروع حتى لا ينكسر التصميم.

## التشغيل
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

إذا ظهر ETIMEDOUT من npm، المشكلة اتصال/registry وليست من ملفات التصميم.
