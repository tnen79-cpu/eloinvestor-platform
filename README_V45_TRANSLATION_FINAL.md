# v45 Translation Final

## المطلوب تشغيله
شغّل الملف التالي في Supabase SQL Editor:

`SUPABASE_V45_TRANSLATION_FINAL.sql`

## ماذا تم في هذه النسخة
- توحيد نظام الترجمة عبر `lib/i18n.ts`.
- إضافة `I18nProvider` حتى تقرأ مكونات الواجهة النصوص من قاعدة البيانات مع fallback داخلي.
- ربط الهيدر، الفوتر، شريط الجوال، وأجزاء من الصفحة الرئيسية بالترجمات.
- إضافة زر مزامنة النصوص الافتراضية داخل لوحة الإدارة.
- إضافة استيراد/تصدير JSON للترجمات.
- SQL آمن ومتوافق مع جدول `platform_languages` القديم الذي يحتوي `name` ومع الجدول الجديد الذي يحتوي `name_ar/name_en`.
- إضافة جدول `translation_provider_settings` لتجهيز مزود ترجمة تلقائية لاحقًا بدون كسر النظام الحالي.

## ملاحظات الفحص
- `npx tsc --noEmit` نجح.
- `npm run build` نجح بالتجميع، ثم طال عند `Collecting page data` كما في النسخ السابقة.
