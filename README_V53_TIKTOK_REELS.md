# v53 TikTok-style Project Reels

## Added
- Fullscreen Reels page without visible public header/footer.
- Snap vertical scrolling like TikTok/Reels.
- Autoplay for the active video.
- Auto-scroll to next video when current video ends.
- Like button with persistent database table.
- Share button using native share/copy fallback.
- Contact button linked to project WhatsApp.
- Project quick link overlay.
- Counters: likes, shares, contacts.
