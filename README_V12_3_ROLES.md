# EloInvestor Next v12.3 Roles Fix

تم تطبيق منطق الأدوار الثلاثة:

- investor: يتصفح الفرص ويتواصل فقط.
- owner: يضيف ويدير مشاريعه فقط.
- both: يجمع صلاحيات المستثمر وصاحب المشروع.
- admin/super_admin: يدخل لوحة الإدارة.

## أهم الإصلاحات

1. إضافة `lib/account.ts` لتوحيد منطق الصلاحيات.
2. منع حساب investor من دخول إضافة مشروع.
3. إخفاء زر إضافة مشروع من قائمة الحساب إذا لم يكن الحساب owner/both/admin.
4. منع حساب owner فقط من التواصل مع المشاريع كمستثمر.
5. إصلاح خطأ خطير: لم يعد `owner` يعتبر admin في لوحة الإدارة.
6. إضافة شارة نوع الحساب في الهيدر ولوحة المستخدم.
7. لوحة المستخدم تعرض صلاحيات المستثمر وصاحب المشروع بوضوح.
