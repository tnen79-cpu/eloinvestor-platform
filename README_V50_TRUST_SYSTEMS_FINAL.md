# v50 Trust Systems Final

تم إعادة ترتيب أنظمة الثقة:

- توثيق المشاريع والمستثمرين يعمل عبر API آمن بدل الاعتماد المباشر على RLS من الواجهة.
- البلاغات تعمل عبر `/api/trust/report` وتظهر في لوحة الإدارة.
- التقييمات تعمل عبر `/api/trust/rating` مع شرط التواصل الفعلي مع المشروع.
- لوحة الإدارة تقرأ البلاغات والتقييمات مباشرة من الجداول بدل الاعتماد على RPC فقط.
- SQL يجهز الجداول والسياسات وBuckets المطلوبة.

## المطلوب
1. شغّل `SUPABASE_V50_TRUST_SYSTEMS_FINAL.sql`.
2. تأكد من وجود `SUPABASE_SERVICE_ROLE_KEY` في `.env.local` لتجاوز مشاكل RLS في الـ API.
3. أعد تشغيل المشروع.
