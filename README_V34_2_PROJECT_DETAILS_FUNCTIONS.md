# v34.2 Project Details Functions

تم تطبيق إصلاحات صفحة تفاصيل المشروع:

- حذف العناصر المكررة في أعلى الصفحة مثل كرت صاحب المشروع وأزرار المشاركة/الحفظ من الهيرو.
- نقل الحفظ والمشاركة إلى Sidebar بشكل أصغر.
- استبدال زر تواصل/استثمر بأزرار: واتساب، اتصال، دردشة.
- إظهار نسبة التمويل فقط إذا كان نوع المشروع تمويل.
- تعريب نوع المشروع: بيع، شراكة، تمويل، امتياز تجاري.
- تقصير النص الطويل في الهيرو ومنع تكراره بصرياً.
- إضافة جدول project_documents وربطه بعرض الوثائق.
- إضافة view للإدارة لعرض وثائق المشاريع.
