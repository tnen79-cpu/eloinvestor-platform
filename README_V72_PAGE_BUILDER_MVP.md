# v72 Page Builder MVP

المضاف:
- تبويب جديد في لوحة الإدارة: **باني الصفحات**.
- إنشاء أقسام الصفحة الرئيسية بدون كود.
- سحب وإفلات لترتيب الأقسام + أزرار أعلى/أسفل.
- إظهار/إخفاء القسم.
- تعديل عربي/إنجليزي للنصوص.
- دعم أقسام: Hero, Search, Sectors, Featured Projects, Banner, Stats, CTA.
- الصفحة الرئيسية تقرأ الأقسام المنشورة من جدول `page_sections`.
- إذا لم توجد أقسام، تبقى الصفحة القديمة كما هي.

شغّل SQL:
`SUPABASE_V72_PAGE_BUILDER_MVP.sql`

ملاحظات:
- هذه نسخة MVP آمنة وليست Elementor كامل.
- يمكن توسيعها لاحقًا لإضافة Blocks أكثر وUpload صور ونسخ/لصق أقسام.
