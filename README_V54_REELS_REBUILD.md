# v54 Reels Rebuild

هذه النسخة تعيد بناء صفحة الريلز بشكل كامل:

- صفحة Full Screen مثل TikTok/Reels.
- بدون هيدر وفوتر.
- تشغيل تلقائي للفيديو الحالي.
- سكرول Snap عمودي.
- انتقال تلقائي للفيديو التالي بعد النهاية.
- لايك / مشاركة / تواصل.
- قراءة الفيديوهات من جدول `project_videos` مباشرة.
- فلترة مرنة حتى لا تختفي الفيديوهات إذا كان `status` أو `country_code` فارغاً في البيانات القديمة.

## مهم

شغّل ملف SQL:

`SUPABASE_V54_REELS_REBUILD_SAFE.sql`

ثم افتح:

`/om/ar/reels`

## سبب المشكلة السابقة

الفيديوهات موجودة في جدول:

`project_videos.video_url`

بينما بعض الإصدارات كانت تعتمد على:

`projects.video_url`

أو كانت تشترط `status/country_code` بطريقة تخفي الفيديوهات.
