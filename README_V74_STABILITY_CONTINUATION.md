# v74 Stability Continuation

هذه النسخة مبنية فوق v71 المستقرة بعد إلغاء فكرة باني الصفحات.

## التعديلات
- تثبيت تجربة الموبايل كـ App Shell بدون لمس الراوتات.
- اعتماد bottom-nav واحد فقط وإخفاء mobile-action-dock القديم.
- ترتيب هيدر الموبايل.
- Drawer جانبي للموبايل بدون تداخل.
- الحفاظ على اللون الأزرق وإزالة الأخضر بصرياً بشكل آمن.
- إبراز الأسعار باللون الأحمر.
- إصلاح قراءة شارات التوثيق لتكون واضحة.

## ملاحظات
- لا يحتاج SQL.
- لم يتم إدخال Page Builder أو Elementor.
- لم يتم تغيير منطق المشاريع أو الشات أو الدفع أو التوثيق.
