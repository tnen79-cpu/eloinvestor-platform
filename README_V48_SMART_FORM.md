# v48 Smart Form

## What's new
- Rebuilt Add Project form as a smarter multi-step form.
- Opportunity type cards: sale, partnership, funding, franchise.
- Dynamic financial fields based on opportunity type.
- Smart completion score.
- Auto-save draft in browser localStorage.
- Smart hints based on title/description keywords.
- One-click description generator.
- Mobile-first spacing and contrast fixes for the new form.

## SQL
Run:
`SUPABASE_V48_SMART_FORM.sql`

The SQL is optional but recommended. The form also keeps the existing fallback behavior for old schemas.
