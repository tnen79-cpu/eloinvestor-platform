# V34.7 Promotion Analytics

## التحديثات
- إضافة إحصائيات نتائج الترويج في لوحة المستخدم: المشاهدات، النقرات، وعدد التواصل.
- إضافة نفس الإحصائيات في لوحة الإدارة داخل تبويب طلبات الترويج.
- عند قبول طلب الترويج من الإدارة يتم تحويله مباشرة إلى `active` ويظهر الزر كـ **قيد الترويج** بدل بقاء زر القبول.
- إضافة SQL جديد: `SUPABASE_V34_7_PROMOTION_ANALYTICS.sql` لإضافة أعمدة الإحصائيات ودالة زيادة آمنة بدون race condition.

## ملاحظة مهمة
شغّل ملف SQL في Supabase قبل الاعتماد الكامل على الإحصائيات حتى تُحفظ القيم داخل جدول `promotion_requests`.
