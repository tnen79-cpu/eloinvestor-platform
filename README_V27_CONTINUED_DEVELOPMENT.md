# EloInvestor v27 Continued Development

هذه النسخة مبنية على v26 وتركز على إصلاحات UX مباشرة بدون SQL جديد.

## تم التنفيذ

- جعل زر تبديل اللغة يحافظ على نفس الصفحة الحالية بدل الرجوع للرئيسية.
- جعل تغيير الدولة من الهيدر يحافظ على نفس المسار الحالي قدر الإمكان.
- إصلاح القائمة السفلية للموبايل: لم تعد صفحة إضافة المشروع مفعّلة دائمًا، بل يتم تفعيل الصفحة الحالية فقط.
- تعديل زر الشات العائم في الموبايل ليفتح صفحة الرسائل الحقيقية بدل تبويب داخل الداشبورد.
- تحويل بحث الهيرو في صفحة الفرص إلى فورم فعلي يرسل `q / sector / maxPrice` إلى صفحة النتائج.
- تحويل فلتر السايدبار في صفحة الفرص إلى فورم فعلي بدل عناصر شكلية فقط.
- إضافة دعم فلترة `opportunityType` داخل محرك البحث المحلي.
- توحيد صفحة التوثيق مع تصميم v26 العام.

## ملاحظات تشغيل

- لا يوجد SQL جديد إجباري لهذه النسخة.
- شغّل أولًا: `npm install`
- ثم: `npm run dev`
- قبل رفع Vercel: `npm run build`
