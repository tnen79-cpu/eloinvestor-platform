# v34.1 Project Details Functional QA

هذه النسخة مبنية على v34 وتفعّل وظائف صفحة تفاصيل المشروع بدل التصميم الشكلي فقط.

## المنفذ
- تفعيل زر حفظ المشروع عبر `SaveProjectButton` مع تحويل الضيف لتسجيل الدخول.
- تفعيل زر مشاركة المشروع عبر Web Share API أو نسخ الرابط.
- تفعيل زر التواصل مع صاحب المشروع عبر المحادثة الداخلية.
- إضافة زر “استثمر الآن” للمستخدم غير المالك.
- إظهار أزرار “تعديل المشروع” و“ترويج المشروع” للمالك فقط.
- تفعيل تبويبات تفاصيل المشروع:
  - نبذة عن المشروع
  - المؤشرات المالية
  - مؤشرات الثقة
  - الوثائق
  - الأسئلة
- فتح الوثائق المرفقة من الجداول الحقيقية إن وجدت.
- استمرار عداد المشاهدات عبر `ProjectViewTracker`.
- إظهار المشاريع المشابهة من نفس القطاع فقط، وإخفاء القسم إذا لا توجد بيانات.
- ربط زر عرض بروفايل صاحب المشروع.
- حذف/إخفاء البيانات الوهمية مثل تقييمات ثابتة وموقع افتراضي غير حقيقي.
- الحفاظ على معرض الصور التفاعلي وفتح الصورة عند الضغط.
