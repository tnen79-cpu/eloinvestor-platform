# EloInvestor Next v21 — Professional Rebuild

هذه النسخة مبنية فوق آخر نسخة مستقرة، لكن تم تنظيف الأساس بدل الترقيع العشوائي.

## أهم ما تم في v21

- SQL موحد وآمن للجداول القديمة والجديدة.
- إصلاح نهائي لصلاحيات الإدارة بدون `infinite recursion`.
- تثبيت حساب `na8061104@gmail.com` كـ Super Admin.
- فصل منطق حسابات الإدارة عن المستخدمين العاديين.
- منع ظهور كل المستخدمين كمدراء.
- تشغيل/إيقاف السلايدر من لوحة الإدارة عبر `platform_settings`.
- تثبيت RLS للمستخدمين والمشاريع والمحفوظات والشات والتوثيق.
- إضافة views/RPC للوحة الإدارة بدل الاعتماد على قراءة مباشرة قد تتكسر مع RLS.
- إبقاء المشاريع المرفوضة مخفية من الواجهة العامة، مع دعم معاينة الإدارة.
- توحيد جداول: الباقات، التوثيق، الإشعارات، البلاغات، التقييمات، السلايدر، البنرات، سجلات الإدارة.

## مهم جدًا

شغّل الملف التالي في Supabase SQL Editor:

```sql
SUPABASE_V21_PROFESSIONAL_REBUILD.sql
```

اضغط **Run** وليس **Explain**.

## بعد تشغيل SQL

1. سجل خروج من المنصة.
2. امسح كاش المتصفح أو افتح Incognito.
3. سجل دخول بالإيميل:

```txt
na8061104@gmail.com
```

4. افتح `/admin`.

## أوامر التشغيل

```bash
npm install
npm run dev
```

للإنتاج:

```bash

npm start
```
