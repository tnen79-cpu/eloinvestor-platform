# EloInvestor v29 — Full OpenSooq-style Clean Redesign

هذه النسخة مبنية على طلب: عدم البناء فوق التصميم القديم، وإعادة تصميم المنصة بالكامل باتجاه سوق مفتوح / OLX مع الحفاظ على كروت المشاريع والوظائف الأساسية.

## ما تم تنفيذه
- استبدال `app/globals.css` بالكامل بهوية جديدة نظيفة، بدون طبقات التصميم القديمة.
- إعادة بناء الصفحة الرئيسية بتصميم Marketplace أبيض/أزرق شبيه بالسوق المفتوح.
- الحفاظ على كروت المشاريع الحالية مع تحسين شكلها وحجمها وتباعدها.
- إعادة تنسيق صفحة الفرص، الفلاتر، الكروت، الهيدر، والموبايل.
- إعادة تنسيق صفحات التفاصيل، الإضافة، الدخول، التسجيل، الباقات، التوثيق، والرسائل عبر نفس نظام التصميم.
- إعادة تنسيق لوحة المستخدم بنمط Dashboard نظيف مع Sidebar.
- إضافة قسم ترويج المشاريع داخل لوحة المستخدم: اختيار مشروع، اختيار باقة، إرسال طلب الترويج.
- دعم موبايل عبر Bottom Navigation وفلاتر/كروت مناسبة للشاشة الصغيرة.
- حذف ملفات README القديمة المرتبطة بالنسخ السابقة لتخفيف التشويش داخل المشروع.

## ملاحظات مهمة
- لم يتم تغيير قاعدة البيانات إجبارياً.
- طلبات الترويج تحاول الحفظ في جدول `project_promotion_requests` إن كان موجوداً، وإلا تحفظ إشعاراً كبديل عبر نظام الـ fallback الموجود.
- لتفعيل نظام الترويج بشكل كامل أنشئ جدول promotion لاحقاً أو اربطه بالدفع.

## التشغيل
```bash
npm install
npm run dev
```

## البناء
```bash
npm run build
```

لم يتم تشغيل البناء داخل بيئة ChatGPT لأن مجلد `node_modules` غير موجود و `npm install` لم يكتمل داخل البيئة.
