# EloInvestor Next v13 - Dashboard Parity

هذه المرحلة نقلت هيكل لوحات HTML إلى Next.js بدقة أعلى:

- Sidebar ديناميكي حسب نوع الحساب: مستثمر، صاحب مشروع، أو الاثنين معًا.
- Hero card بنفس روح نسخة HTML وبألوان خضراء هادئة.
- كروت الحساب الأساسية: نوع الحساب، الباقة، رصيد النشر، حالة التوثيق.
- لوحة صاحب المشروع: مشاريعي، إضافة مشروع، طلبات المستثمرين، الباقات، الإحصائيات.
- لوحة المستثمر: مركز المستثمر، المحفوظات، المشاريع التي تواصل معها، الفرص المقترحة، الاهتمامات.
- الحساب المزدوج يعرض كل الأقسام بدون تكرار.
- روابط مباشرة للشات، التوثيق، الباقات، إضافة مشروع، وتفاصيل المشروع.
- جلب بيانات Supabase مع fallbacks للجداول الاختيارية حتى لا تنكسر الواجهة إذا جدول غير موجود.
- build ناجح باستخدام مفاتيح Supabase وهمية.

المرحلة التالية المقترحة: v13.5 لنقل الشات realtime ومركز الإشعارات والطلبات بشكل أعمق داخل نفس layout.
