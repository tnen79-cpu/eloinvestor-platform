# EloInvestor Next v18.1 — Admin QA & Development

## أهم التطويرات

- تحسين لوحة الإدارة كمركز تحكم فعلي وليس واجهة فقط.
- إضافة عدادات تنبيه داخل تبويبات الإدارة للمشاريع/التوثيق/الترقيات/الإشعارات.
- إضافة فلاتر لطلبات التوثيق وطلبات الترقية.
- إضافة بحث للمستخدمين والصلاحيات.
- تسجيل عمليات الإدارة في جدول `admin_action_logs`.
- إرسال إشعارات تلقائية للمستخدم عند:
  - تغيير حالة مشروعه.
  - قبول/رفض/طلب تعديل التوثيق.
  - قبول/رفض طلب الترقية.
  - تعديل بيانات أو صلاحيات حسابه.
- تحسين رسائل النجاح والخطأ بعد الحفظ.
- SQL آمن للتشغيل المتكرر ومتوافق مع الجداول القديمة.
